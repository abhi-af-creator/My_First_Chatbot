# My_First_Chatbot
AI-Powered PDF Chatbot is an intelligent Retrieval-Augmented Generation (RAG) application that allows users to upload PDFs and ask questions about their content. It uses LangChain, FAISS, and HuggingFace models to extract, retrieve, and generate accurate, context-aware answers through a simple Streamlit interface
